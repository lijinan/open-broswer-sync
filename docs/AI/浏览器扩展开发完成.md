## 📱 安装使用指南

### 🔧 Chrome/Edge 安装

#### 开发者模式安装 (推荐)
1. **打开扩展管理**:
   - Chrome: 访问 `chrome://extensions/`
   - Edge: 访问 `edge://extensions/`

2. **开启开发者模式**:
   - 点击右上角"开发者模式"开关

3. **加载扩展**:
   - 点击"加载已解压的扩展程序"
   - 选择 `browser-extension` 文件夹
   - 扩展安装完成！

#### 打包安装
```bash
1. 在扩展管理页面点击"打包扩展程序"
2. 选择 browser-extension 文件夹
3. 生成 .crx 文件
4. 拖拽 .crx 文件到扩展管理页面
```

### 🦊 Firefox 安装

#### 临时安装 (开发测试)
1. **打开调试页面**: 访问 `about:debugging`
2. **选择此Firefox**: 点击"此Firefox"
3. **临时载入**: 点击"临时载入附加组件"
4. **选择清单**: 选择 `manifest-firefox.json` 文件

#### 永久安装
```bash
1. 安装 web-ext 工具: npm install -g web-ext
2. 重命名清单文件: mv manifest-firefox.json manifest.json
3. 打包扩展: web-ext build
4. 安装生成的 .xpi 文件
```

## 🎮 使用教程

### 🚀 首次设置

1. **安装扩展**: 按照上述方法安装扩展
2. **点击图标**: 点击浏览器工具栏的扩展图标
3. **配置服务器**: 确认服务器地址 (默认: http://localhost:3001)
4. **登录账号**: 输入用户名和密码登录
5. **开始使用**: 登录成功后即可使用所有功能

### 📖 保存书签

#### 方法一: 扩展弹窗
```
1. 点击扩展图标
2. 点击"保存当前页面为书签"
3. 确认保存 (如开启确认提示)
4. 查看成功通知
```

#### 方法二: 右键菜单
```
1. 右键点击页面空白处
2. 选择"保存为书签"
3. 确认保存
4. 查看成功通知
```

### 🔐 保存密码

#### 自动检测模式
```
1. 访问有登录表单的网站
2. 观察表单附近的"💾 点击保存密码"提示
3. 点击提示快速保存
4. 或填写表单后提交，会自动弹出保存提示
```

#### 手动检测模式
```
1. 在登录页面点击扩展图标
2. 点击"检测并保存密码"
3. 扩展会自动检测表单内容
4. 确认保存密码信息
```

#### 右键保存模式
```
1. 在登录页面右键点击
2. 选择"保存密码信息"
3. 扩展会检测并提示保存
4. 确认保存
```
