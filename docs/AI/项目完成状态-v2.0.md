## 🎮 使用指南

### 🌙 主题切换
1. **自动检测**: 首次访问自动检测系统主题偏好
2. **手动切换**: 点击顶部工具栏主题开关
3. **快捷键**: 按 `Ctrl+,` 快速切换主题
4. **持久化**: 主题选择自动保存，下次访问保持

### ⌨️ 快捷键操作
```
🌐 全局快捷键:
Ctrl+N  新建项目 (根据当前页面)
Ctrl+F  聚焦搜索框
Ctrl+H  返回仪表板首页
Ctrl+E  跳转导入导出页面
Ctrl+,  切换明暗主题
?       显示快捷键帮助
Esc     取消操作或失去焦点

🧭 页面导航:
Alt+1   仪表板
Alt+2   书签管理  
Alt+3   密码管理
Alt+4   导入导出

📋 列表操作:
j       下一项
k       上一项
1-9     选择对应项目
Shift+Enter  打开选中项目
```

### 📤 数据导入导出
1. **导入格式**: 支持JSON、CSV、HTML格式
2. **导出格式**: 支持JSON格式完整导出
3. **浏览器兼容**: 支持Chrome、Firefox书签导入
4. **重复检测**: 自动检测和处理重复数据
