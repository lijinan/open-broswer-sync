## 🔧 新的开发规范

### 文件存放规则
1. **技术文档** → `docs/AI/`
2. **扩展测试** → `browser-extension/test/`
3. **项目测试** → `test/`
4. **核心文档** → 根目录

### 命名规范
- **测试文件**: `test-功能名称.html/js`
- **调试文件**: `debug-功能名称.js`
- **文档文件**: `功能描述.md`

### 开发工作流
1. **新功能开发** → 在对应目录创建测试文件
2. **问题修复** → 在 `docs/AI/` 记录修复过程
3. **测试验证** → 使用相应目录的测试文件
